# 🐾 PataBot — الوكيل الذكي الشامل لـ PataHogar.com

## 🚀 What is PataBot?
PataBot is a comprehensive AI-powered dropshipping agent for **patahogar.com** — a pet & home products store powered by BigBuy.

## ✨ Features
- **📦 Product Management** — Fetches profitable products from BigBuy API, calculates margins (30-50%), updates inventory
- **📢 Marketing** — Creates content for Meta, TikTok, Snapchat. Organic + Paid ads (with owner approval)
- **🔍 Research** — Analyzes market trends, Ad Library insights, winning products
- **💬 Customer Service** — Auto-replies in 8 languages, order tracking, returns
- **🔒 Security** — 24/7 website monitoring, SSL checks, backups
- **📊 Reports** — Daily reports sent to owner via email

## 🌍 Supported Languages
Spanish, English, French, German, Dutch, Italian, Arabic, Russian

## 🛠 Tech Stack
- Python 3.11 + FastAPI
- BigBuy REST API
- Meta Business API
- TikTok Business API
- Hosted on Railway

## 📋 API Endpoints
- `GET /` — Bot status
- `GET /api/dashboard` — Control panel
- `GET /api/products` — Current products
- `GET /api/products/fetch` — Fetch new products
- `GET /api/products/trending` — Trending products
- `POST /api/chat` — Chat with PataBot
- `GET /api/report/daily` — Daily report
